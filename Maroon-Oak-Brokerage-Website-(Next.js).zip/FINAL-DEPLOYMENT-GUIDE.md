# Maroon Oak Brokerage — Final Website Build

This production-ready Next.js website includes the premium homepage, personal insurance, business insurance, claims, resources, about, contact, privacy, and terms pages.

## Deploy
1. Extract the ZIP.
2. Upload the **contents** of this folder to the root of the GitHub repository.
3. Commit and push to `main`.
4. In Vercel, use Framework Preset **Next.js**, leave Root Directory blank, and redeploy without cache.

## Before launch
- Connect the quote/contact form to the final email or CRM endpoint.
- Replace carrier wordmarks only with carrier-approved assets and show only markets that Maroon Oak is authorized to represent.
- Add live social profile URLs and licensing disclosures.
- Review all policy, privacy, and legal language with counsel/compliance.

The file `public/design-reference-homepage.png` is included only as the approved visual reference and is not rendered by the site.
